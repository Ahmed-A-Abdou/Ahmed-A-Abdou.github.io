Claude Code 101

The Course
Claude Code 101 is Anthropic's official introductory course on AI-assisted software development using Claude Code — an agentic coding tool that operates directly in the terminal and IDE. The course is aimed at developers new to AI coding agents and covers everything from installation to building custom workflows with subagents, skills, and MCP servers. Estimated completion time: ~1 hour.
What It Covers
The course is built around a single repeatable development rhythm — Explore → Plan → Code → Commit — and covers 10 modules:
  • What Claude Code is and how the agentic loop works (context window, tools, permissions)
  • Installing Claude Code across terminal, VS Code, JetBrains, and Claude Desktop
  • The Explore → Plan → Code → Commit workflow for real development tasks
  • Context management — /compact, /clear, and long-session hygiene
  • Code review workflows with Claude Code
  • The CLAUDE.md file — project memory that persists across sessions
  • Subagents, Skills, and MCP servers for connecting external tools
  • Hooks — deterministic guardrails controlling what Claude can and can't do
What I Took Away
The course formalised how I use Claude Code day-to-day — particularly the Explore → Plan phase before writing any code, and the CLAUDE.md system for keeping project context consistent across sessions. The hooks and MCP sections introduced tooling I now use to keep Claude Code's behaviour predictable and auditable on real projects.
