Unity Pathway Unit 3 Side-Scroller 3D

3D Side-Scrolling Runner

About this project
Unit 3 prototype from the Junior Programmer Pathway — a 3D side-scrolling endless runner where the character moves forward automatically while the player dodges obstacles. The project brings together obstacle spawning, a scrolling background, particle effects, and character animations into a complete endless runner loop.
What I Built
  • 3D side-scrolling camera setup with automatic forward movement
  • Obstacle spawning system with timed intervals
  • Scrolling background using repeating world segments to create an infinite feel
  • Player jump mechanic with responsive controls
  • Particle effects on running and collision
  • Character animations tied to gameplay states — running, jumping, falling
  • Collision detection triggering game over
  • Sound effects for jumping, colliding, and background music
What I Learned
This project taught me how to create the illusion of infinite movement through world scrolling rather than actually moving the player. Hooking up the Animator Controller to gameplay states showed me how animation transitions need to be driven by code cleanly — not just triggered but properly conditioned. Working with particle systems for the first time made me appreciate how much visual feedback contributes to game feel.
